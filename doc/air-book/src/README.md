# AIR Tool Documentation

The AI Robustness (AIR) Tool allows users to gauge AI/ML classifier performance with unprecedented confidence by applying causal inference methods to detect confounding bias in predictive models.

This book is the complete reference for understanding, installing, and operating the AIR Tool. It is intended for both technical practitioners working with AI/ML models and domain experts contributing subject-matter knowledge to the analysis process.

## How to Use This Book

If you are new to the AIR Tool, start with the [Introduction](01-introduction.md) and [Understanding Causal Thinking](04-causal-thinking.md) before moving into the operational sections.

If you are ready to get the tool running, go to [Getting Started](02-getting-started.md) and [Installation](03-installation.md).

If you are actively running an analysis, the [Running the AIR Tool](07-running/README.md) section walks through each step of the workflow in detail.

## About the AIR Tool

The AIR Tool was developed by the Software Engineering Institute (SEI) at Carnegie Mellon University as part of the TAILOR project. It employs Causal Discovery and Causal Inference to determine when an AI's classifications and predictions are confounded by bias — helping answer the critical question of whether an AI is ready to deploy.
